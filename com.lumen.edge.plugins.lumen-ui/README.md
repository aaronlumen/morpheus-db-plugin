# Lumen UI Plugin
# - com.edge.lumen.plugins.lumen-ui                      - #

# - gradle assemble
# - gradle build
# - enjoy
# - - - - - - - - - - - - - - - - - - - - - - - - - - - - -#
This plugin provides Lumen specific customizations to the Morpheus UI including:
- SalesForce Chat Bug

## Build
Build the JAR file
```shell
./gradlew shadowJar
```

Jars should be placed in 
``` 
/var/opt/morpheus/morpheus-ui/plugins
```

## Test
```shell
./gradlew test
```
